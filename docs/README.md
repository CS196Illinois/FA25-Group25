# Public Website Files

**IMPORTANT: This folder contains files for the PUBLIC GitHub Pages website!**

## Public URL
https://cs196illinois.github.io/FA25-Group25/

## How to Update the Public Site

1. **Edit files in THIS folder** (`docs/`), NOT the `124H website/` folder
2. Stage your changes: `git add .`
3. Commit your changes: `git commit -m "your message"`
4. Push to GitHub: `git push`
5. Wait 1-2 minutes for GitHub Pages to deploy
6. Hard refresh your browser (Ctrl+Shift+R or Cmd+Shift+R)

## Key Files
- `index.html` - Main page
- `main.css` - Main stylesheet (linked from index.html)
- `(2)html.html` - Current ForEX page
- `(3)html.html` - ForExplorer page
